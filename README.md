# PanControl

Sistema de Gestión para Panaderías - Control de inventario, producción, ventas y reportes.

## Características

- Dashboard con métricas en tiempo real
- Gestión de productos e insumos
- Control de producción con consumo automático de insumos
- Punto de venta (POS) con carrito
- Reportes de ventas y producción
- Gestión de empleados con roles (admin/empleado)
- Facturación e impresión de tickets
